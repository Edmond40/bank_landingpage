# MerchantSpring Bank

Premium international banking frontend with official brand logo and navy + royal blue theme.

## Pages

- **/** — Landing page
- **/receipt** — International wire transfer receipt
- **/about** — Company history and leadership
- **/contact** — Contact form and branch locations

## Tech Stack

- React 19 + Vite
- Tailwind CSS v4
- React Router
- Framer Motion
- React Icons

## Getting Started

```bash
npm install
npm run dev
```

## Brand Assets

Logo: `public/logo.png`

## Build

```bash
npm run build
npm run preview
```
